[WinDbg](https://learn.microsoft.com/windows-hardware/drivers/debuggercmds/windbg-overview) is a multipurpose debugger for the Microsoft Windows computer operating system, distributed by Microsoft. It can be used to debug user mode applications, device drivers, and the operating system itself in kernel mode.
